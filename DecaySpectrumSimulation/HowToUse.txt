
для запуску програми достатньо запустити 'main.py' через інтерпретатор python 3
це можна зробити наступним чином: перейти в командний рядок, перейти до директорії з файлами проекту і прописати
python main.py


файл 'input.txt': тут вказується початкові дані, якими заповнюються поля у програмі. У процесі виконання програми
цей файл не змінюватиметься. Ви можете відредагувати його, підставивши свої параметри.
файл 'main.py': файл з кодом реалізації gui програми
файл 'plot_figure.py': файл із реалізацією класу алгоритму моделювання апаратурного спектру
файл 'spectr.txt': вихідний файл із набором точок спектру


gui
поле peaks:
тут вводяться дані піків, такі як енергія (E, (kev)) в kev, інтенсивність I (інтенсивність визначається як аплітуда піку),
ширена на половині висоти FWHM в kev (визначає ширину піку)
-Для додавання нового піку, достатньо ввести його характеристики у відповідних осередках
порожній рядку
-для видалення піку, потрібно залишити порожню комірку відповідної енергії піку
-при неправильному введенні даних (негативний)

поле background:
визначає фон спектру двома функціями: лінійною (y=a*x+b) та експоненційною (y=a*exp(-b*x))
кожна приймає в полі введення по два параметри 'a' та 'b'

поле range Energy:
Emax поле - число рівне максимальної енергії вказане на графіку вказане в Kev
num channels поле - кількість каналів, на яке розбивається діапазон енергій (від 1 до Emax)
bias поле - число (Kev), що вказує зміщення графіка щодо осі каналів (X)

для оновлення графіків із новими параметрами потрібно натиснути кнопку 'update'

після натискання кнопок fig 1 - fig 4 малюється спектр обраного етапу в окремому вікні
